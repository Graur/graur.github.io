# Personal web page based on GitHub Pages
https://graur.github.io/
---

### Run

```bash
$ bundle update
$ bundle exec jekyll serve
```

### Installation

```bash
$ gem install jekyll bundler
$ jekyll new personal-blog
$ bundle exec jekyll serve --watch
```
---

### Deployment

```bash
$ npm run develop
$ npm run build
```
